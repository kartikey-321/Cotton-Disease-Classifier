# Cotton-Disease-Prediction

It classifies whether an image is of diseased cotton leaf, diseased cotton plant, fresh cotton leaf or fresh cotton plant

## Technologies Used
1. HTML
2. CSS
3. JavaScript
4. Deep Learning (Transfer Learning)
5. Flask
